Custom Roles
------------

* A role based on an existing role.

  .. role:: custom(literal)

  :custom:`one` :custom:`two` :custom:`three`

* A new role.

  .. role:: customnew

  :customnew:`one two three`

* A role with class attribute.

  .. role:: customclass
     :class: special

  :customclass:`interpreted text`
   
* A role with class attribute.

  .. role:: customx
     :class: very special 

  :customx:`interpreted text`
