SYNOPSIS
========

``rst2man`` ``--help``

``rst2man`` [ OPTIONS ] [ SOURCE [ *destination* ] ]


DESCRIPTION
===========

Run it and examine output.

OPTIONS
=======

-o x   an option
-b     another

For all other options see ``--help``.


EXAMPLES
===========

:rst2man.py xml-schema-catalog.rst xml-schema-catalog.man:
    create a manpage from xml-schema-catalog.rst
