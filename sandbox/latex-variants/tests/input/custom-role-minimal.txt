The "role" directive dynamically creates a custom interpreted text
role and registers it with the parser.  

After declaring a role, the document may use the new "custom" role for
`interpreted text`. It is possible to set several class names, for example:

.. role:: custom
   :class: custom custom2 my_custom custom.role

Big, large, small-caps :custom:`Interpreted Text` with a custom role.
